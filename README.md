Standard Node.JS TS App which download data from amazon server, parse them to flat JSON array structure and save them to local storage.

## Prerequisites

```
node
npm
```

## To install

`npm install`

## To run fetch&save

`npm start`

## To test

`npm test`

## To run just fetching data

`npm run fetch`

## To run just storing to local storage

`npm run save`
